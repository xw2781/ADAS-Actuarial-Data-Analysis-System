# Release 0.2.3

Released on 2026-05-07.

## User-Facing Changes

### Improvements
- **build**: Installer progress now advances during the long ArcRho package extraction step instead of staying at 10% until completion.
  - The Windows installer now uses an ArcRho extraction helper that reads 7-Zip progress and updates the installer details and progress bar while the embedded application package is unpacked.
- **shell**: Server Connection changes now apply without restarting the app.
  - Saving a new root path refreshes runtime paths and notifies open views so cached path data is cleared.
  - Ordinary page reloads no longer send app shutdown, preventing a Server Connection save from quitting the app.
